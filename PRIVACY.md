# Privacy Policy

## Overview

*Context Menu mklink* is an *explorer.exe* extension that create symbolic links and hard links in file system of your computer. This software is committed to protecting your privacy.

## Data Collection

This software **does not collect any personal data**. Specifically:

* No user data is collected or transmitted to any servers
* No analytics or tracking tools are used
* No operation history is recorded
* No personal information is stored

## Data Usage

The software only:

* Read the file path in your clipboard
* Create link files on your computer
* Ask privilege to create link files if current permission is not enough
* All processing happens entirely on your device

## Privilege

The software may ask for privilege to create link files, only if user triggers.

## Third-Party Services

This software does not use any third-party services or send data to any external servers.

## Data Storage

All files are stored locally on your device. The data never leaves your hardware.

The update dates can be seen at the top-right corner of this page.
